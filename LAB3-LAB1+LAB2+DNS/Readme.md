LAB3-LAB1+LAB2+DNS

목표
- LAB1과 LAB2를 통합하고, 간단한 DNS를 구현한다.
- 내부 PC에서 내부 서버 및 외부 서버에 Domain name으로 접속이 가능하게 한다.


사용할 기술 OSPF, NAT, DHCP, DNS, ROAS
-------

고려할 점
- LAB1과 LAB2를 토폴로지 통합하려 한다.
- 기반은 LAB2로 하되, LAB1에서 사용된 기술을 추가한다.
- LAB1에선 따로 DHCP 서버를 사용했고, LAB2에선 라우터를 DHCP 서버로 사용했는데, LAB3에선 라우터DHCP를 제거하고, LAB1처럼 DHCP 서버를 따로 사용한다.
- 또한 LAB2는 간단한 default 및 static route를 사용했지만, LAB3에선 LAB1처럼 OSPF를 사용한다.
- LAB1처럼 내부에서 외부 인터넷(8.8.8.8) 및 서버로 나갈땐 NAT 변환한다.
- 방화벽은 개념적으로 설치해놨지만, 따로 정책을 구현하진 않는다.
